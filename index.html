Catalyzer

Catalyzer is a tweet analyzer that could predicts user's interests. It receives tweets via Twitter API and categorize them to four different topics: technology, entertainment, sports, and health & beauty. The analysis is done by string matching algorithm: Knuth-Morris-Pratt (KMP) and Boyer-Moore

ZIP SMS

SMS application that can encrypt text and signed it with digital signature. Encrption is done using a block-cipher algorithm, while the digital signature is built using the combination of ECDSA algorithm and SHA-1 hash function. The digital signature is represented as hexadecimals at the end of the text. This application also enables users to decrypt and authenticate incoming messages.

  • Tools: Java, Android Studio
  • Date: December 2016
